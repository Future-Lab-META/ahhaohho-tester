## 개요

아하오호 챌린지에는 '정답'이 없지만, 처음 접하는 유저는 "어떻게 만들어야 하지?"라는 막막함을 느끼기 쉽다. 챌린지 피처드 프로젝트는 실제 또래 아이가 만든 우수 프로젝트를 챌린지별 참고 예시로 지정해 보여주는 것으로, 직접 지시하지 않고 좋은 예시를 통해 창작의 방향을 자연스럽게 안내한다.

---

## 문제
챌린지 진입 시 참고할 수 있는 예시가 없어 창작 방향을 잡기 어려움. 특히 신규 유저일수록 "이런 걸 만들면 되는구나"를 느낄 레퍼런스가 없어 이탈 가능성이 높음

## 우리의 해결책
체험단 운영 중 운영자가 어드민에서 챌린지 게시물을 조회하고, 퀄리티 있는 게시물을 피처드 프로젝트로 직접 지정. 챌린지 상세 화면 상단 썸네일 영역에 화살표 버튼을 추가해, 썸네일 옆으로 피처드 프로젝트를 넘겨볼 수 있도록 함.

**노출 구조**
- 첫 번째 슬라이드: 챌린지 썸네일 (고정)
- 이후 슬라이드: 피처드 풀 전체 중 최대 10개 랜덤 순서로 노출 (피처드 지정 개수 제한 없음)
- 예시: 썸네일 → 피처드5 → 피처드3 → 피처드2 → 피처드1 → 피처드4
- 영상은 자동 재생(loop), 사운드 기본값 off (on 불가)
- 피처드가 없는 챌린지는 화살표 버튼 미노출, 기존 썸네일만 표시

**데이터 로드 기준**
- 피처드 목록은 챌린지 상세화면 진입 시점 기준으로 로드
- 화면을 나갔다가 재진입 시 새로 조회 (세션 내 캐시 없음)
- 커뮤니티 외부·내부 동일하게 노출

**화살표 버튼**
- 첫 번째 슬라이드(썸네일): 우측 화살표만 노출
- 중간 슬라이드: 좌우 화살표 모두 노출
- 마지막 슬라이드: 좌측 화살표만 노출
- 탭 또는 스크롤 모두 슬라이드 이동 동작

**메이커 닉네임 표시**
- 피처드 슬라이드에 한해 이미지 상단에 "{닉네임}" 레이블 노출
- 챌린지 썸네일 슬라이드에서는 미노출

**프로젝트 이미지·영상**
- 피처드 게시물의 첫 번째 미디어부터 순서대로 노출 (이미지 또는 영상)
- 다수의 미디어 프로젝트일 경우 모든 페이지에 샛별 아이콘 동일하게 노출

**샛별 아이콘 태그**
- 앱 노출 명칭: 샛별
- 에셋: 아이콘만 (텍스트 없음)
- 노출 위치: 이미지·영상 우측 상단
- 탭 시 별도 동작 없음
- 피처드 취소 시 태그 제거 타이밍: 화면을 나갔다가 재진입 시 새로 조회 기준과 동일 (세션 내 캐시 없음)
- 다수의 미디어 프로젝트일 경우 모든 페이지에 동일하게 노출

**갤러리 피드 노출**
- 갤러리 메인 피드에서도 피처드 프로젝트에 샛별 아이콘 동일하게 표시
- 노출 위치·에셋 기준은 챌린지 상세와 동일 (이미지·영상 우측 상단, 아이콘만)
- 다수의 미디어 게시물일 경우 모든 페이지에 동일하게 노출

**운영 흐름**
1. 운영자가 어드민 프로젝트 관리에서 챌린지 게시물 조회
2. 선정 기준 검토 후 피처드 지정 (해제 가능)
3. 지정된 피처드는 앱 챌린지 상세 화면에 자동 반영

## 알림

**발송 조건**
- 피처드 지정 시 해당 프로젝트 게시자에게 자동 발송

**알림 포맷**
> {챌린지명} 챌린지의 샛별로 선정됐어요! 축하해요 🎉

**알림 정책**
- 피처드 취소 시 이미 발송된 알림에 영향 없음 (최초 발송 메시지 유지)
- 알림 탭 시 해당 챌린지 상세 화면 진입 후 샛별 슬라이드로 바로 이동
- 해당 화면 진입 후 이탈할 경우 새로운 알림은 모두 읽음 상태로 전환 (모든 알림 공통)

**알림 발송 시점 표시**
- 방금 전 (1시간 이내) / 1시간~23시간 / 1일~6일 / 1주~

## 기대 효과
- 챌린지 진입 후 게시 완료율 향상
- 실제 또래 프로젝트를 통한 "나도 할 수 있겠다"는 동기 부여
- 선정된 아이에게 인정받는 경험 제공

## KPI
- 챌린지별 피처드 프로젝트 확보율 (4기 종료 시 전체 챌린지 커버)
- 피처드 프로젝트 노출 챌린지 vs 미노출 챌린지 게시 완료율 비교

## 원칙
1. 피처드 프로젝트는 실제 또래 아이의 프로젝트만 — 운영자가 직접 만든 예시 사용하지 않음
2. 선정 기준(창의성·완성도·영감 가능성)을 아이들에게 사전 공개 — "왜 저 프로젝트가 뽑혔지?"라는 의문이 생기지 않도록 함
